## VKvmd
VKontakte voice messages downloader - Загрузчик голосовых сообщений ВКонтакте

Простой способ скачать большое количество голосовых сообщений из ВКонтакте за раз, будь то личный диалог или беседа, без использования сторонних сервисов. Этот способ я использовал, когда тренировал голосовые модели с помощью ИИ.

## Инструкция по использованию
1. Первое, что нам необходимо сделать это запросить архив с вашими диалогами ВКонтакте. Переходим по [ссылке](https://vk.com/data_protection?section=rules&scroll_to_archive=1) и делаем запрос, архив с перепиской, как правило, будет готов уже через пару часов. Скачиваем и распаковываем в удобное место.
2. Далее нам необходимо узнать id пользователя, чьи голосовые сообщения мы хотим скачать. В браузере откройте личный диалог или беседу, нажмите на аватар участника. В адресной строке появится `peer0123456789`. Число после `peer` является искомым id. Находим и открываем в скачанном архиве нужную нам папку с id.
3. В адресной строке проводника наберите `cmd` и нажмите Enter.
Командой `ren *.html *.txt` преобразуем файлы в .txt:
```
ren *.html *.txt
```
Командой `copy *.txt messages.txt` объединяем все страницы диалога в один файл:
```
copy *.txt messages.txt
``` 
4. В адресной строке замените `cmd` на `powershell` или наберите `powershell` после открытия `cmd`.  
Вставьте скрипт, заменив `0123456789` на id собеседника:
```
$id=0123456789;mkdir -Force "voice_messages_$id"|Out-Null;sls "vk\.com/doc$id" messages.txt|%{[regex]::Match($_.Line,"https:[^']+\.ogg").Value}|?{$_}|select -Unique|%{$f=Split-Path -Leaf $_;write-host "Загрузка $f";iwr $_ -OutFile "voice_messages_$id\$f"}
```
Для скачивания голосовых сообщений другого пользователя из той же беседы достаточно заменить id и выполнить скрипт снова, предыдущие шаги повторять не нужно.  

5. Файлы .ogg сохранятся в подпапку `voice_messages_$id`.
6. [ОПЦИОНАЛЬНО] Удаление .txt файлов из папки:
```
Remove-Item -Path *.txt -ErrorAction SilentlyContinue
```
