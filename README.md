## VKvmd - VKontakte voice messages downloader
### Достаточно простой способ скачать большое количество голосовых сообщении из ВКонтакте за раз, будь то личный диалог или беседа, без использования сторонних сервисов. Этот способ я использовал когда тренировал голосовые модели с помощью ИИ.

## personal.ps1 & conference.ps1
### Инструкция по использованию
1. Первое что нам необходимо сделать это запросить архив с вашими диалогами ВКонтакте. Переходим по [ссылке](https://vk.com/data_protection?section=rules&scroll_to_archive=1) и делаем запрос, архив с единственными диалогами как правило будет готов уже через пару часов. Скачиваем и распаковываем в удобное место.
2. Далее нам необходимо узнать id пользователя голосовые с которым мы собираемся скачать. Это можно сделать зайдя в диалог с нужным нам пользователем или беседой, последние цифры в адресной строке являются нужным нам id. Открываем в скачанном архиве нужную нам папку с id.
3. Активируем консоль `cmd` в пути с открывшейся папкой. Командой `ren *.html *.txt` преобразуем файлы в .txt. Командой `copy *.txt messages.txt` объединяем все страницы диалога в один файл.
4. После этого переходим в PowerShell, аналогично с cmd, копируем нужный скрипт для [личного диалога](https://github.com/caefolomy/VKvmd/blob/main/personal.ps1) или [беседы](https://github.com/caefolomy/VKvmd/blob/main/conference.ps1). Единственное что вам необходимо сделать, это заменить заглушку [путь_к_каталогу] на путь к нашей папке в первой строке кода. Внимательно читайте комментарии в коде, особенно в [conference.ps1](https://github.com/caefolomy/VKvmd/blob/main/conference.ps1)!
5. Файлы скачаются по указанному выше пути с расширением .ogg.
